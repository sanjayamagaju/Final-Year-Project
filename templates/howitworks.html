Welcome to "Life Savior"

-->This is how our System works<--

Before accessing the platform users have to register their account if they are new otherwise, they can login to the system and access the platform. To create their account, they have to provide username, email, password which will be saved in the database after checking their validity which helps them to login for next time easily.

After successfully completing account register and login process, user will be redirected to the home page. They can view the running campaigns, donate certain amount using the Khalti API and also, they can create the campaign if needed. Only that user who has created the campaign can update or delete the campaign. Other users can only view and has no chance of editing and deleting the campaigns. On clicking the victim’s name, the user will be redirected to the detail page where they can see only that clicked campaign. Also, user can search for certain campaign by searching for the victim’s name or description of that campaign.

There is a donate button in every campaigns. When user/donors click on that button it will redirect them to a page where they will be appreciated for donating along with the donate using Khalti button. On clicking this button, they will be asked the amount to donate and after putting amount they will be redirected to the Khalti API where they have to login to their Khalti account using phone number and password. If the account is valid Khalti will send OTP code. Using this code, users can successfully donate on that campaign.

There is another page named Donation for future purpose alongside the donation for campaign page. This page is for collecting some donation amount for future purpose when some victim might need huge money suddenly like millions of dollars which is hard to complete on few days. This page is I believe one of the best features which no other donation platform provides.
